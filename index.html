Bubble Sort

Bubble sort is a method of ordering a list of items from biggest to smallest or from smallest to biggest, the example on the right of the screen orders the bars from biggest to smallest.

At each step the algorithm compares two of the bars, it will swap the bars if the one on the left is smaller than the one on the right, you can click the next button to see this in action.

The algorithm will move through the whole list swapping items until it reaches the end, at the end it will check to see if the list is sorted, if it is sorted then the algorithm is finished, if not it will go from the beggining of the list again, this will repeat until the list is fully sorted.

Insertion Sort

Bubble sort is a method of ordering a list of items from biggest to smallest or from smallest to biggest, the example on the right of the screen orders the bars from biggest to smallest.

Insertion sort starts with the second value in the list, if the value is smaller than the one on its left, it stays where it is, othewise it will be swapped left until it meets a value bigger than it or it reaches the start of the list.

The algorithm then repeates this process with the next value, and then the next and the next until it has reached the end of the list, once it has reached this point, the list is sorted.
